## Featured Project

### Munmai — Financial Clarity System

Munmai is a production-deployed MERN application designed to help users understand where their money is going, what they owe, what others owe them, and how imported financial data should be reviewed before it changes their records.

**Live App:** https://munmai.com  
**Repository:** https://github.com/kshitijchaudhary/munmai  

### What Munmai includes

- Income and expense tracking
- Receipt tracking
- Monthly spending control
- Debt Reality for loans, credit cards, bills, and borrowed money
- CSV Import Review Queue
- Shared group expenses
- Group invitations and join codes
- Settlements and balance calculations
- Monthly Summary reports
- Opening Balance setup
- JWT authentication and protected user data

### Tech Stack

React, Vite, Node.js, Express.js, MongoDB, Mongoose, JWT, Axios, Vercel, Render, MongoDB Atlas
